## ✨ Fonctionnalités

- ⚡️ Nuxt 3 + Vue 3
- 🎨 Nuxt UI Components
- 🎯 Responsive Design
- 🌗 Dark/Light Mode

## 🚀 Installation

```bash
# Installation des dépendances
yarn install

# Lancement du serveur de développement
yarn dev

# Build production
yarn build
```
